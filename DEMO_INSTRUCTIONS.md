# Sasaya AI - Demo Instructions


Sasaya AI Mac App 的 MVP 版本已經建立完成，包含完整的 UI 和 mock data。

## 📦 Phase 1 已實現的功能

### ✅ 核心功能
1. **三欄式 UI 佈局**
   - 左側欄：分類和筆記列表
   - 主內容區：筆記編輯器
   - 右側 AI 對話區

2. **分類管理**
   - 12 個預設分類（包含 emoji 圖示）
   - 展開/收合功能
   - 筆記數量顯示

3. **筆記管理**
   - 10 個 mock 筆記範例
   - 選擇筆記查看
   - 即時編輯標題和內容
   - 時間戳記顯示

4. **搜尋功能**
   - 頂部搜尋欄
   - 即時關鍵字搜尋
   - 搜尋標題和內容

5. **編輯工具列**
   - 格式化按鈕（粗體、斜體、底線等）
   - ChatGPT 整合按鈕
   - 複製、分享、刪除功能

6. **AI 對話區**
   - Sasaya AI 助手介面
   - 搜尋範圍選擇器
   - 對話輸入框
   - 歡迎訊息

## 🚀 如何啟動

### 方法 1: 使用 start 腳本（推薦）
```bash
cd /Users/alexho/Projects/SasayaByCursor1
npm start
```

### 方法 2: 分別啟動
```bash
# 終端 1
npm run dev

# 終端 2（等待 Vite 啟動後）
npm run electron
```

### 方法 3: 在瀏覽器預覽
```bash
# 如果 Vite 已經在執行
# 直接打開瀏覽器訪問：
open http://localhost:5173
```

## 🔍 測試功能

### 1. 瀏覽筆記
- 點擊左側分類展開/收合

- 點擊筆記標題編輯
- 在主內容區編輯筆記內容

- 可以選擇搜尋範圍
- 輸入框準備好接收問題

### 框架和工具
- **Electron**: Mac 桌面應用程式
- **React 19**: UI 框架
- **Vite**: 快速開發伺服器
- **純 CSS**: 無額外 UI 框架

### 檔案結構
```
SasayaByCursor1/
├── electron/
│   └── main.js              # Electron 主程序
├── src/

│   ├── App.jsx              # 主應用程式

```
